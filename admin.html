Edit Gallery

Upload photos, move them between categories and reorder them directly on this page. Then export gallery-data.js and re-upload the site to Netlify so everyone sees the changes.

How it works:

Changes save in this browser first. To publish them online: click Export gallery-data.js, replace the old file in your website folder, zip the folder and deploy to Netlify.
